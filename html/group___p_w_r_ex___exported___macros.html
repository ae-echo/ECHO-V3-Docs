Definition stm32h563xx.h:14541

Configure the main internal regulator output voltage.

Note
This macro is similar to HAL_PWREx_ControlVoltageScaling() API but doesn't check whether or not VOSREADY flag is set. User may resort to __HAL_PWR_GET_FLAG() macro to check VOSF bit state.
Parameters
__REGULATOR__: Specifies the regulator output voltage to achieve a tradeoff between performance and power consumption. This parameter can be one of the following values :
  • PWR_REGULATOR_VOLTAGE_SCALE0 : Regulator voltage output scale 0. Provides a typical output voltage at 1.2 V. Used when system clock frequency is up to 160 MHz.
  • PWR_REGULATOR_VOLTAGE_SCALE1 : Regulator voltage output scale 1. Provides a typical output voltage at 1.1 V. Used when system clock frequency is up to 100 MHz.
  • PWR_REGULATOR_VOLTAGE_SCALE2 : Regulator voltage output scale 2. Provides a typical output voltage at 1.0 V. Used when system clock frequency is up to 50 MHz.
  • PWR_REGULATOR_VOLTAGE_SCALE3 : Regulator voltage output scale 3. Provides a typical output voltage at 0.9 V. Used when system clock frequency is up to 24 MHz.
Return values
None.
