Check PWR flags are set or not.

Parameters
__FLAG__: Specifies the flag to check. This parameter can be one of the following values :
  • PWR_FLAG_STOPF : Stop flag. Indicates that the device was resumed from Stop mode.
  • PWR_FLAG_SBF : Standby flag. Indicates that the device was resumed from Standby mode.
  • PWR_FLAG_VOSRDY : Voltage scaling ready flag. Indicates that the Vcore level at or above VOS selected level.
  • PWR_FLAG_ACTVOSRDY : Currently applied VOS ready flag. Indicates that Vcore is equal to the current voltage scaling provided by ACTVOS.
  • PWR_FLAG_BRR : Backup regulator ready flag. This bit is not reset when the device wakes up from STANDBY mode or by a system reset or power-on reset.
  • PWR_FLAG_VBATL : Backup domain voltage level flag (versus low threshold). Indicates the backup domain voltage level is equal or above low threshold.
  • PWR_FLAG_VBATH : Backup domain voltage level flag (versus high threshold). Indicates the backup domain voltage level is equal or above high threshold.
  • PWR_FLAG_TEMPL : Temperature level flag (versus low threshold). Indicates the temperature is equal or above low threshold.
  • PWR_FLAG_TEMPH : Temperature level flag (versus high threshold). Indicates the temperature is equal or above high threshold.
  • PWR_FLAG_AVDO : Regulator selection flag. Indicates the regulator selected.
  • PWR_FLAG_VDDIO2RDY : VDDIO2 ready flag (versus 0.9 V threshold). Indicates that VDDIO2 is equal or above the threshold of the VDDIO2 voltage monitor (around 0.9 V).
  • PWR_FLAG_PVDO : Voltage detector output flag. Indicates that Vdd is equal or above the PVD threshold selected by PVDLS.
  • PWR_FLAG_USB33RDY : VDDUSB ready flag (versus 1.2 V threshold). Indicates that VDDUSB is equal or above the threshold of the VDDUSB voltage monitor (around 1.2 V).
  • PWR_WAKEUP_FLAG1 : Wakeup flag 1. Indicates that a wakeup event was received from the WKUP line 1.
  • PWR_WAKEUP_FLAG2 : Wakeup flag 2. Indicates that a wakeup event was received from the WKUP line 2.
  • PWR_WAKEUP_FLAG3 : Wakeup flag 3. Indicates that a wakeup event was received from the WKUP line 3.
  • PWR_WAKEUP_FLAG4 : Wakeup flag 4. Indicates that a wakeup event was received from the WKUP line 4.
  • PWR_WAKEUP_FLAG5 : Wakeup flag 5. Indicates that a wakeup event was received from the WKUP line 5.
  • PWR_WAKEUP_FLAG6 : Wakeup flag 6. Indicates that a wakeup event was received from the WKUP line 6.
  • PWR_WAKEUP_FLAG7 : Wakeup flag 7. Indicates that a wakeup event was received from the WKUP line 7.
  • PWR_WAKEUP_FLAG8 : Wakeup flag 8. Indicates that a wakeup event was received from the WKUP line 8.
Note
The PWR_WAKEUP_FLAG6, PWR_WAKEUP_FLAG7 AND PWR_WAKEUP_FLAG8 are not available for STM32H503xx devices.
Return values
Thestate of FLAG (TRUE or FALSE).
