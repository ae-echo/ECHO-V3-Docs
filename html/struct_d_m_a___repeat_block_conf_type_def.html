◆ BlkDestAddrOffset

int32_t BlkDestAddrOffset

Specifies the DMA channel block destination address offset signed value : This parameter can be a value between -65535 and 65535. If block destination address offset > 0 => Increment the block destination address by offset from where the last block ends. If block destination address offset < 0 => Decrement the next block destination address by offset from where the last block ends. If block destination address offset == 0 => the next block destination address starts from where the last block ends
