Detailed Description

Note
HAL I3C option value coding follow below described bitmap: b31 0 : message end type restart 1 : message end type stop b30-b29-b28-b27 0010 : I3C private message 0011 : direct CCC message 0110 : broadcast CCC message 0100 : I2C private message b4 0 : message without arbitration header 1 : message with arbitration header b0 0 : message without defining byte 1 : message with defining byte

other bits (not used)
